# 🛒 Retail Sales Performance Dashboard
### Built with Tableau Desktop | Indian E-Commerce Data | 2021–2024

---

## 📌 Project Overview

This project presents an **end-to-end retail sales analytics dashboard** built in Tableau, analyzing **1,000 retail transactions** across India over 4 years (2021–2024). It covers sales performance, profitability, customer behavior, regional trends, and payment insights — all in one interactive dashboard.

> 🎯 Built as part of a Data Analytics practical assignment and included in my professional portfolio.

## 💡 Key Insights from the Dashboard

- 📦 **Electronics** is the top-selling category with ₹5.2M in revenue
- 📱 **UPI** is the most preferred payment mode (28% of orders)
- 🏙️ **Mumbai & Delhi** are the highest-performing cities by sales + profit
- 📉 Peak sales months (Oct–Nov) show **compressed profit margins** due to discounting
- ⭐ **Corporate** customers give consistently higher ratings than other segments
